Authenticates a PhilAm Life Reward

The PhilAm Life Rewards API is one of the PhilAm Application programming interfaces (APIs) that grants the application owner permission to use to data of all reward of the PhilAm life when invoking reward system. This API utilizes OAuth 2.0 authorization framework that requires the account holder to login using his/her credentials, select an account, verify via One-Time Password, and provide consent to debit their account.

What are the key features?

  • Displays a web view of the PhilAM Customer Authentication page where the PhilAM account holder is required to input the credentials, select the account for the transaction, and grant consent to the application owner.
  • Prevents unauthorized access to the PhilAm Life client’s account.
  • Enables authentication and linking of account through short-lived or long-lived access tokens
  • An application that hold data about client premium and info can secure consent from its PhilAm account holder.
